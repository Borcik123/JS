  1. kiedy powstał JavaScript (podaj datę).

    JavaScript został stworzony w 1995 roku.

  2. Dlaczego jest to język skryptowy.

    Jest to język skryptowy, ponieważ jest interpretowany w czasie rzeczywistym, a nie kompilowany wcześniej jak języki kompilowane. Skrypty są uruchamiane po stronie klienta lub serwera.

  3. Gdzie jest wykonywany JS (klient lub serwer).

    Tradycyjnie JavaScript był wykonywany po stronie klienta (w przeglądarkach internetowych). Jednak z pojawieniem się technologii takich jak Node.js, może być również wykonywany po stronie serwera.

  4. Dlaczego JS to bezpieczne narzędzie.

    JavaScript jest uważany za bezpieczny, ponieważ działa w kontrolowanym środowisku przeglądarki i nie ma bezpośredniego dostępu do systemu operacyjnego komputera. Jednak, jak każde narzędzie, może być używany do celów niepożądanych, jeśli nie jest prawidłowo napisany lub zabezpieczony.

  5. Czy jest to język obiektowy, jeśli tak to dlaczego.

    Tak, JavaScript jest językiem obiektowym. Umożliwia tworzenie, dziedziczenie i manipulację obiektami. Choć ma również cechy języków funkcyjnych.

  6. Wymień zalety JS:


    Zaleta1
    Jest wszechstronny i można go używać zarówno po stronie klienta, jak i serwera.
    Zaleta2
    Integracja z technologiami internetowymi, takimi jak HTML i CSS.
    Zaleta3
    Duże wsparcie społeczności i wiele dostępnych bibliotek i frameworków.

  7. Framework to:

    Framework to zestaw narzędzi, bibliotek i najlepszych praktyk, które ułatwiają tworzenie aplikacji według określonej architektury lub wzorca.

  8. Do aplikacji mobilnych można użyć Framework.

    Tak, istnieje wiele frameworków JavaScript, takich jak React Native czy Ionic, które umożliwiają tworzenie aplikacji mobilnych.

  9. Aplikacje responsywne to:

    Aplikacje responsywne to takie, które są projektowane w taki sposób, aby dostosowywać się do różnych rozmiarów ekranów i urządzeń, zapewniając optymalne doświadczenie użytkownika.

  10. Przepisz linie kody pod nimi wytłumaczenie jak działa ta ten fragment kodu.

    input type="button" name="przycisk" value="Nowa Strona" onclick="WinOpen(' ')">
    Ten fragment kodu tworzy przycisk, który po kliknięciu próbuje otworzyć nowe okno (poprzez niezdefiniowaną funkcję "WinOpen"), ale obecnie adres URL jest pusty.

  11. Przepisz linie kody pod nimi wytłumaczenie jak działa ta ten fragment kodu.

    window.open("obraz.html","okienko","toolbar=no,directories=no,menubar=no,height=280,width=160,top=200,left=200");

  12. Opisz następującą instrukcję window.close()

    Instrukcja window.close() zamyka bieżące okno przeglądarki lub zakładkę. Warto zauważyć, że niektóre przeglądarki mogą blokować to działanie, chyba że okno zostało otwarte za pomocą skryptu.
